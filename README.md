The analysis utilizes the Healthcare Cost Report Information System (HCRIS) data, specifically merging the v1996 and v2010 forms.

It seeks to investigate the causal relationship between federal Medicare penalties and private insurance pricing. Specifically, it looks at whether hospitals offset financial losses from the Hospital Readmissions Reduction Program (HRRP) and Value-Based Purchasing (VBP) programs by increasing the prices they charge to private insurers (cost shifting).
